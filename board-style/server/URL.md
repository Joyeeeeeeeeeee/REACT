# SpringBoot REST API
- 목록 : /boards                [GET]
- 조회 : /boards/{no}           [GET]
- 등록 : /boards                [POST]
- 수정 : /boards                [PUT]
- 삭제 : /boards/{no}           [DELETE]

# React 라우팅      
- 메인 : /                      Home
- 목록 : /boards                board/List
- 조회 : /boards/{no}           board/Read
- 등록 : /boards/insert         board/Insert
- 수정 : /boards/update/{no}    board/Update
